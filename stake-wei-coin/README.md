# stake-wei-coin

## Description
- 質押wei-coin可以獲得reward
- reward根據stake數量,時間(block-height)來計算


## How To Run This Project

### Run

```shell
# check contract  
$ clarinet check 
# test contract in local 
$ clarinet test    
```

### Other

```shell
# create new contract  
$ clarinet contract new <contract-name>    
```
